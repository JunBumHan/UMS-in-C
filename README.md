# UMS-By-C

### COMMIT MESSAGE NAMESPACE

|Index |Explanation|
|------|----------------------|
|ADD:: |기능을 추가할 때 사용합니다.|
|SUP:: |기능을 보완할 때 사용합니다.|
|FIX:: |기능을 수정할 때 사용합니다.|
|DEL:: |기능을 삭제할 때 사용합니다.|
|RAN:: |커밋 명세가 귀찮을 때 사용합니다.|
|UPD:: |ADD, FIX, DEL에 맞혀 현재 상황을 최신 상황으로 수정할 때 사용합니다.|

> 수정하다[FIX::] : 고치어 정돈하다.

> 보완하다[SUP::] :모자라거나 부족한 것을 보충하여 완전하게 하다.
